# Getting Started

### Config swagger to access the API documentation
Step 1: Add the following dependency to the build.gradle file.

Step 2: Edit file Application main class and add the following annotation @Import(SpringDocConfiguration.class)

Step 3: Go to the browser and access the following url: http://localhost:8080/swagger-ui.html to access the API documentation.
